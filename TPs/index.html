TP : Protocole HTTP - Support Complet (Énoncé & Corrigé)

TP 1 : Exploration avec les DevTools

1.1 Ouvrir les DevTools

1. Ouvrez Chrome ou Firefox. 2. Appuyez sur F12. 3. Allez dans l'onglet Network.

1.2 Observer une requête simple

Naviguez vers https://httpbin.org/get et observez la requête.

Réponses aux questions :
• Code de statut : 200 OK.
• Headers de requête : User-Agent, Host, Accept.
• Content-Type : application/json.

TP 2 : Maîtrise de cURL

2.1 Requêtes de base

# Requête basique
curl https://httpbin.org/get

# Avec headers de réponse
curl -i https://httpbin.org/get

# Mode verbose (complet)
curl -v https://httpbin.org/get
Différence entre -i et -v :
-i affiche uniquement les headers de la réponse. -v affiche tout le processus (handshake, headers envoyés et reçus).

Exercice avancé : Requête POST complexe

curl -i -X POST \
     -H "Content-Type: application/json" \
     -H "X-Custom-Header: MonHeader" \
     -d '{"action": "test", "value": 42}' \
     https://httpbin.org/post

TP 3 : API REST & JavaScript

Exercice pratique : fetchWithRetry

async function fetchWithRetry(url, options, maxRetries) {
  for (let i = 0; i <= maxRetries; i++) {
    try {
      const response = await fetch(url, options);
      if (response.status >= 500) throw new Error(`Erreur ${response.status}`);
      return response;
    } catch (err) {
      if (i === maxRetries) throw err;
      console.warn(`Tentative ${i+1} échouée. Nouvel essai dans 1s...`);
      await new Promise(res => setTimeout(res, 1000));
    }
  }
}
