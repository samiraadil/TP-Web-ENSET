TP : Le Protocole HTTP - Énoncé & Corrigé

TP 1 : Exploration avec les DevTools

1.1 Ouvrir les DevTools

  1. Ouvrez Chrome ou Firefox
  2. Appuyez sur F12 ou Ctrl+Shift+I
  3. Allez dans l'onglet Network (Réseau)

1.2 Observer une requête simple

  • Cochez "Preserve log" pour garder l'historique
  • Naviguez vers https://httpbin.org/get
  • Cliquez sur la requête dans la liste

Questions : Quel est le code de statut ? Quels headers de requête sont envoyés ? Quel est le Content-Type de la réponse ?

Réponses :
  • Code de statut : 200 OK (Indique que la requête a réussi).
  • Headers de requête : On y trouve User-Agent, Accept, Host, et Accept-Encoding.
  • Content-Type : application/json (Le serveur renvoie des données au format JSON).

TP 2 : Maîtrise de cURL

2.1 Requête GET simple

# Requête basique
curl https://httpbin.org/get

# Avec headers de réponse
curl -i https://httpbin.org/get

# Mode verbose (debug)
curl -v https://httpbin.org/get
Question : Quelle est la différence entre -i et -v ?
Réponse : L'option -i affiche uniquement les headers de la réponse avant le contenu. L'option -v (verbose) est un mode de debug qui affiche tout l'échange : la connexion IP, les headers que tu envoies (marqués par >) et ceux que tu reçois (marqués par <).

Exercice avancé : Requête POST

TP 3 : API REST avec JavaScript

Exercice pratique : fetchWithRetry

Créez une fonction fetchWithRetry(url, options, maxRetries) qui gère les erreurs 5xx et attend 1 seconde entre chaque tentative. 
async function fetchWithRetry(url, options, maxRetries) {
  for (let i = 0; i <= maxRetries; i++) {
    try {
      const response = await fetch(url, options);
      
      // Si c'est une erreur serveur (500, 502, 503, 504)
      if (response.status >= 500 && response.status < 600) {
        throw new Error(`Erreur serveur : ${response.status}`);
      }
      
      return response; // Retourne la réponse si succès (2xx) ou erreur client (4xx)

    } catch (err) {
      const isLastAttempt = i === maxRetries;
      if (isLastAttempt) {
        console.error("Nombre maximal de tentatives atteint.");
        throw err;
      }
      
      console.warn(`Tentative ${i + 1} échouée (${err.message}). Nouvel essai dans 1s...`);
      // Attendre 1000ms avant la prochaine itération
      await new Promise(resolve => setTimeout(resolve, 1000));
    }
  }
}
