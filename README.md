# TP Développement Web

## Objectif
Réaliser une page web en HTML et CSS.

## Technologies utilisées
- HTML : structure de la page
- CSS : mise en forme

## Contenu du site
- Header avec navigation
- Section Accueil
- Section À propos
- Formulaire de contact
- Footer

## Fonctionnalités
- Navigation entre sections
- Affichage d’image
- Formulaire simple

## Conclusion
Ce TP m’a permis de comprendre les bases du développement web et la séparation entre structure (HTML) et style (CSS).
